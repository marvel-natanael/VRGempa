# VRGempa

## Description
This is an earthquake simulation tested on Oculus Quest 2

## Usage
1. Click on one of the buttons to start simulation
2. Nod your head down
3. Point the controllers backwards as if you are protecting your head
4. If you have performed the required action, the result will be Green.
